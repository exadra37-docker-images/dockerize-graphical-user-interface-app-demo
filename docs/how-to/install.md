# HOW TO INSTALL


#### Git Clone Project

```bash
$ git clone https://gitlab.com/exadra37-docker/productivity-tool/go-for-it.git
```

#### Enter Project Dir

```bash
$ cd go-for-it
```

#### Confirm Content

```bash
$ ls -1
AUTHOR.md
build
composer.json
CONTRIBUTING.md
CONTRIBUTORS.md
dgad
docs
LICENSE
README.md
src
```

#### Check Permissions

```bash
 ls -al dgad
-rwxr-xr-- 1 exadra7 exadra7 1401 Mar 12 14:47 dgad
```

>**NOTE**: if the bit `x` is missing in the file permissions, please add it:
>
>```bash
>$ chmod ug+x dgad
>```
